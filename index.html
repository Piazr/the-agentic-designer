About the Book

The Agentic Designer covers the full breadth of the agentic design paradigm: agent platforms, design-for-agents harnesses, design-as-code workflows, and multi-agent systems orchestration.

Written with AI assistance using an agentic publishing pipeline. The author directed, reviewed, and approved all content. Technical review by Claude Opus 4.6 and Gemini 3.1 Pro. Free for personal use under a personal use license.

The book includes 14 chapters, 3 appendices with battle-tested prompts, copy-paste code examples, and four real-world case studies with honest assessments of what worked and what fell apart. Whether you’re a designer curious about AI agents or an engineer building design tools, this book provides the mental models and practical workflows you need.

“Agent choice matters less than agent configuration. A well-configured agent with a good skill file will outperform a bare agent every time.”
